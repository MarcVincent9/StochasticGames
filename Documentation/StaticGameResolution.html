StaticGameResolution module¶

StaticGameResolution.maximin(game, expected_rewards, state, playerA, playerB)¶

solve the static game associated with one state of a null sum 2-player stochastic game

Parameters:
  • game – NullSum2PlayerStochasticGame
  • expected_rewards – dictionary {player B action : {player A action : reward}}
  • state – state ID
  • playerA – maximizing player ID
  • playerB – minimizing player ID
Return type:

state value (float) and player A strategy (dictionary: {action: probability})
